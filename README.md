# College-Student-Management-System


A full-stack web application built using Spring Boot, React.js, MySQL, and JWT Authentication.

# Features

Secure User Authentication (JWT Based)

Role-Based Authorization (Admin & Teacher)

Student Management (Add, Update, Delete, View)

Marks Management

Fee Management

Dashboard Analytics

RESTful API Architecture

Modern Responsive UI

# Tech Stack
Backend

Java 17

Spring Boot

Spring Security

JWT (JSON Web Token)

Hibernate / JPA

MySQL

Frontend

React.js

Axios

React Router

Custom CSS

# Security Features

Password Encryption using BCrypt

Stateless Authentication

Role-Based Endpoint Protection

JWT Token Validation Filter
